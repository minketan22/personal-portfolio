# Minketan Portfolio

A modern personal portfolio website built to showcase projects, skills, and contact information for internships, freelance work, and frontend developer opportunities.

## Overview

This portfolio highlights:

- A premium glassmorphism-inspired landing page
- A recruiter-friendly introduction and personal summary
- Skills, highlights, and selected project sections
- Direct contact options and resume access
- A responsive layout for desktop and mobile devices

## Built With

- HTML5
- CSS3
- JavaScript
- AOS Animation Library
- Font Awesome

## Sections Included

- Hero section
- About
- Skills
- Highlights
- Featured project
- Project showcase
- Contact section

## Getting Started

To run this project locally:

1. Clone the repository
2. Open the project folder in VS Code
3. Open `index.html` in your browser

## Deployment

This project can be deployed easily using GitHub Pages.

1. Push the project to a GitHub repository
2. Open the repository on GitHub
3. Go to `Settings > Pages`
4. Under `Build and deployment`, choose `Deploy from a branch`
5. Select the `main` branch and `/root`
6. Save the settings

After deployment, GitHub Pages will generate your live site URL.

## Customization

You can personalize this portfolio by updating:
